# @amlplugins/twilio-video

Native `twilio` SDK re-exported under the `@amlplugins` namespace.

> Twilio Programmable Video — Rooms, Compositions, recordings, real-time video collaboration.

## Install

```bash
npm install @amlplugins/twilio-video
```

## Use

```ts
import { /* named exports from twilio */ } from "@amlplugins/twilio-video";
// or
import sdk from "@amlplugins/twilio-video";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
